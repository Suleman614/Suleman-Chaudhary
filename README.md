# Suleman Chaudhary – Data Science & Analytics Portfolio

This is the source code for my personal portfolio website built with **Astro**.  
The site showcases my work in **data science, analytics, predictive modeling**, and my academic experience as an **Information Systems major at The Ohio State University**.

---

## Tech Stack
- Astro  
- TailwindCSS  
- TypeScript / JavaScript  
- Markdown  
- GitHub Pages

---

## Development
To run the site locally:

```bash
npm install
npm run dev
```

The site will be available at:
http://localhost:4321
